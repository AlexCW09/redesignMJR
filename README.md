## Objective
The objective for the MJR app redesign was to enhance the user experience by improving the navigation, design elements, and features. It addresses the existing usability issues, aims to increase user engagement, and be visually appealing with a more modern design. By priortizing user-centered design principles and conducting user testing, the goal is to make the app more user-friendly.
